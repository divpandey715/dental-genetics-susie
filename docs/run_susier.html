Overview

This analysis applies SuSiE-RSS and mvSuSiE-RSS fine-mapping to dental GWAS summary statistics and locus-specific LD matrices.

The GWAS summary statistics used in this project come from Shungin et al., “Genome-wide analysis of dental caries and periodontitis combining clinical and self-reported data,” published in Nature Communications in 2019. This study analyzed dental caries and periodontal traits using clinical measures from the GLIDE consortium and self-reported oral health traits from UK Biobank. The clinical dental traits used in this fine-mapping workflow include DMFS, DFSS, and Nteeth.

Article: Shungin et al. 2019, Nature Communications

The current workflow is implemented for DMFS, Nteeth, and DFSS and includes:

Single-trait SuSiE-RSS

  1. SuSiE credible set results
  2. Positional mapping to nearest hg19 genes
  3. eQTL colocalization summary

Joint mvSuSiE-RSS

  1. DMFS + Nteeth
  2. DMFS + DFSS
  3. Nteeth + DFSS
  4. DMFS + Nteeth + DFSS

For joint mvSuSiE-RSS analyses, only variants with PIP = 1 are displayed, with nearest hg19 gene annotation added from the positional mapping results.

make_dt <- function(data, caption_text) {
  datatable(
    data,
    extensions = "Buttons",
    caption = caption_text,
    options = list(
      dom = "Blfrtip",

        c(10, 25, 50, "All")
      ),
      scrollX = TRUE
    )
  )
}

make_nearest_genes <- function(...) {
  bind_rows(...) %>%
    dplyr::select(
      MarkerName,
      external_gene_name,
      ensembl_gene_id,
      any_of(c("distance_to_gene", "distance", "distance_to_nearest_gene"))
    ) %>%
    distinct(MarkerName, .keep_all = TRUE)
}

Joint mvSuSiE-RSS Fine-Mapping

Joint analysis workflow

For the joint mvSuSiE-RSS analyses, I used the same general workflow across all trait pairs and the three-trait analysis:

  1. GWAS summary statistics were aligned to the same locus-specific EUR LD matrices.
  2. The corresponding .id_map files were used to map LD rsIDs to chromosome-position marker names.
  3. Z-scores were calculated as Effect / StdErr.
  4. mvSuSiE-RSS was run using the same prior variance structure and L = 10.
  5. Joint PIP, credible set assignment, purity, Z-scores, effect estimates, standard errors, p-values, sample sizes, and nearest hg19 gene annotation were summarized.

For all joint mvSuSiE-RSS analyses, only variants with PIP = 1 are displayed in the tables below.

The mvSuSiE-RSS scripts are available here:

  • DMFS + Nteeth: 07_mvsusie_dmfs_nteeth.R
  • DMFS + DFSS: 11_mvsusie_dmfs_dfss.R
  • Nteeth + DFSS: 12_mvsusie_nteeth_dfss.R
  • DMFS + Nteeth + DFSS: 13_mvsusie_dmfs_nteeth_dfss.R
