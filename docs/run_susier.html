Overview

This analysis applies SuSiE-RSS fine-mapping to dental GWAS summary statistics and LD matrices.

The current workflow is implemented for DMFS and includes:

  1. SuSiE credible set results
  2. Positional mapping to nearest hg19 genes
  3. eQTL colocalization summary
